# 📂 Testing Types

This section covers Functional and Non-Functional testing methodologies used to verify software.

## 🗂️ Sub-Directories

### 1. [Functional Testing](./functional/)
Testing the software against the functional requirements/specifications.
- [Unit Testing](./functional/01_unit_testing.md)
- [Integration Testing](./functional/02_integration_testing.md)
- [System Testing](./functional/03_system_testing.md)
- [UAT / Acceptance Testing](./functional/04_uat_acceptance_testing.md)
- [Smoke Testing](./functional/05_smoke_testing.md)
- [Sanity Testing](./functional/05_smoke_testing.md) (Redirect)
- [Regression Testing](./functional/07_regression_testing.md)

### 2. [Non-Functional Testing](./non_functional/)
Testing the software characteristics like performance, security, usability, etc.
- [Performance Testing](./non_functional/01_performance_testing.md)
- [Security Testing](./non_functional/06_security_testing.md)
- Load, Stress, Spike, Soak, Usability, Compatibility, Accessibility, and Recovery Testing.
